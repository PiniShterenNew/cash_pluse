# CashPulse — Screen Map & Build Order

## Priority 0 (MVP core — build first)
1. **Landing Page** — Hero + pain/relief + features + pricing + FAQ. Marketing page.
2. **Login / Signup** — Split layout, Google auth, friendly copy.
3. **Onboarding (3 steps)** — Business info → balance/expenses → import invoices.
4. **Dashboard** — KPIs (4 cards) + cash forecast chart + overdue list + upcoming events + quick actions.
5. **Receivables Board (Kanban)** — 5 columns: הונפקה, מגיע בקרוב, באיחור, בגבייה, שולם. Drag & drop.
6. **Cash Flow Timeline** — Area+bar combo chart + transactions table + danger zone alerts.
7. **WhatsApp Reminder Modal** — Template picker (3 tones) + bubble preview + send.
8. **Invoice Create/Edit Modal** — Client autocomplete, amount, dates, payment terms.
9. **Empty States** — Every screen needs a warm, encouraging empty state.
10. **Mobile Views** — Bottom nav, swipe kanban, card-view tables.

## Priority 1 (after MVP launch)
11. **Client Profile** — Reliability gauge + stats + invoice history + communication log.
12. **Reports & Analytics (Pro)** — Monthly summary + client bubble chart + collection funnel.
13. **Settings** — Profile, WhatsApp config, alerts, import/export, plan management.
14. **CSV Import Flow** — Drag & drop + column mapping + preview.
15. **Notifications Panel** — Dropdown from bell, grouped by date.
